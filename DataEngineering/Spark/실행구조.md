- application
  - 여러개의 Job으로 구성
  - RDD에 어떤 액션을 수행하면 job 생성(Ex.write() 같은 함수 호출 시)
- job
  - 여러개의 stage로 구성
  - 셔플을 필요로 하는 함수가 있으면 새로운 stage 생성됨. 
  - 데이터셋을 읽을 때도 새로운 stage 생성
- stage
  - 여러개의 task로 구성
  - stage내의 Task들은 모두 동일한 명령을 수행
- task
  - spark에서 가장 작은 작업 단위
  - 일련의 명령을 실행(ex. reading data, filtering, ...)
